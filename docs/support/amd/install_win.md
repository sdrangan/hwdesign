## Windows-Specific Instructions

After registering the AMD account, you can download the unified installer for windows:



Open the installer

<img src="figs\welcome.png" alt="welcome" style="zoom:75%;" />

Enter your account and password:

<img src="figs\login.png" alt="login" style="zoom:75%;" />

To install the complete Vitis and Vivado, select Vivado to install and then add-on vitis:

<img src="figs\select_vivado.png" alt="select_vivado" style="zoom:75%;" />

Select the standard version

<img src="figs\select_standard.png" alt="select_standard" style="zoom:75%;" />

Select device(SoCs & 7 Series) to install

<img src="figs\select_devices.png" alt="select_device" style="zoom:75%;" />

Agree all:

<img src="figs\agrees.png" alt="agrees" style="zoom:75%;" />

Set your install location

<img src="figs\install_location.png" alt="install_location" style="zoom:75%;" />

Check the Installation Summary:

<img src="figs\summary.png" alt="summary" style="zoom:75%;" />

Start the Installation, will take a while:

<img src="figs\installing.png" alt="installing" style="zoom:75%;" />

When it prompts to add the license, download the license file on the website

<img src="figs\llicens_manager.png" alt="llicens_manager" style="zoom:75%;" />

<img src="figs\license.png" alt="license" style="zoom:75%;" />

Click "Load License" in the left bar

<img src="figs\load_lic.png" alt="load_lic" style="zoom:75%;" />

And then select "copy the licenses", where you select the downloaded "Xilinx.lic" file.

Successfully Installedï¼š

<img src="figs\success.png" alt="success" style="zoom:75%;" />
